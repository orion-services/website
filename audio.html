Percepção Sonora - Audio

Synchronous System Notification Sheet Music

Below are the scores of the sound signals developed to be used in a synchronous system for people with visual impairments with the intention of seeking better virtual interaction and accessibility.

ALT 001

Sound used to notify shortcut number 1 call.


ALT 002

Sound used to notify shortcut number 2 call.


ALT 003

Sound used to notify shortcut number 3 call.


CNX 001

Sound to notify that the user is logged into the system.


CTB 001

This sound is called when a user initiates their input to the text.


CTB 002

This sound is called when a user finishes their contribution to the text.


HTR 001

This sound is called when the user enables the history of contributions.


HTR 002

This sound is called when the user advances through the history of contributions.


HTR 003

This sound is called when the user goes back in the history of contributions.


MSG 001

This sound plays when the user sends a message.


MSG 002

This sound plays when the user receives a message.


RBC 001

This sound plays when a participant agrees that a goal has been achieved.
